🖥️ Mini OS Simulator

A modern Operating System Simulator developed using Python and Tkinter. This project provides an interactive graphical environment for visualizing and understanding fundamental Operating System concepts through simulation and real-time visualization.

✨ Features
⚙️ CPU Scheduling
First Come First Serve (FCFS)
Shortest Job First (SJF)
Priority Scheduling
Round Robin (RR)
Gantt Chart Visualization
Performance Metrics Analysis
🧠 Memory Management
First Fit
Best Fit
Worst Fit
Next Fit
Paging Simulation
Fragmentation Analysis
Memory Allocation Visualization
📄 Page Replacement
FIFO

🔄 Process Synchronization
