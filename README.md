# 🖥️ Mini OS Simulator

A modern Operating System Simulator developed using **Python** and **Tkinter**. This project provides an interactive environment for visualizing and understanding fundamental Operating System concepts through simulation and real-time visualization.

---

## ✨ Features

### ⚙️ CPU Scheduling
- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Priority Scheduling
- Round Robin (RR)
- Gantt Chart Visualization
- Performance Metrics Analysis

### 🧠 Memory Management
- First Fit
- Best Fit
- Worst Fit
- Next Fit
- Paging Simulation
- Fragmentation Analysis
- Memory Allocation Visualization

### 📄 Page Replacement
- FIFO
- LRU
- Optimal
- Page Fault & Hit Analysis
- Frame State Visualization

### 🔄 Process Synchronization
- Producer-Consumer Problem
- Semaphore Simulation
- Mutex Lock Mechanism
- Real-Time Buffer Visualization

### 🔒 Deadlock Handling
- Banker's Algorithm
- Safe Sequence Detection
- Resource Allocation Analysis

### 💾 File Management
- Sequential Allocation
- Linked Allocation
- Indexed Allocation
- Disk Block Visualization
